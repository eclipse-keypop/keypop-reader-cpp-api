Keypop Reader C++ API Documentation

Overview

The Keypop Reader C++ API defines interfaces for reader communication aligned with the Terminal Reader specifications from the Calypso Networks Association. This C++ interface provides a port of the reference Keypop Reader Java API.

Architecture & Components

Basic Reader Operations

  • keypop::reader::CardReader Core interface for basic card reader operations and state
  • keypop::reader::ConfigurableCardReader Extended interface for configurable card readers with protocol management

Card Selection Management

  • keypop::reader::selection::CardSelectionManager Service for preparing and executing card selection scenarios
  • keypop::reader::selection::CardSelector Common card filtering interface for selection process
  • keypop::reader::selection::CardSelectionResult Container for card selection operation results

Card Reader Observation

  • keypop::reader::ObservableCardReader Interface for observable reader features and card detection
  • keypop::reader::CardReaderEvent Card insertion/removal event information
  • keypop::reader::spi::CardReaderObserverSpi Interface for card reader event observation

ISO Card Support

  • keypop::reader::selection::IsoCardSelector Selection filters for ISO 7816-4 cards
  • keypop::reader::selection::spi::IsoSmartCard Interface representing selected ISO 7816-4 cards

Extension Support

  • keypop::reader::selection::spi::CardSelectionExtension SPI for card-specific selection extensions
  • keypop::reader::selection::spi::SmartCard Base interface for smart card representation

Exception Handling

The API implements the following exception hierarchy:

  • keypop::reader::CardCommunicationException Card communication failure handling
  • keypop::reader::ReaderCommunicationException
    Reader communication failure handling
  • keypop::reader::ReaderProtocolNotSupportedException Unsupported protocol error handling
  • keypop::reader::selection::InvalidCardResponseException Invalid card response handling during selection
